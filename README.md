## การติดตั้ง
Clone โปรเจคให้เรียบร้อย และอย่าลืมลง Package "pyamaze"ลงสู่โปรเจค
ลง Package โดยการไปที่ Terminal ของโปรเจคแล้วพิมพ์คำสั่ง
```sh
pip install pyamaze
```
